Introduction

Welcome to the Shiro Optimizer documentation.

This tool is a comprehensive Python-based utility designed to optimize, debloat, and secure Windows 10 and 11 systems. It combines powerful batch scripts, PowerShell automation, and trusted third-party tools into a unified "Hacker-style" interface.

‚öôÔ∏è Tweaks Explained

Below is a detailed breakdown of every script found in the tweaks/ directory.

basicstuff.bat

Applies fundamental Explorer preferences and visual effects optimizations.

Bloatware.bat

Aggressively removes OEM bloatware and unnecessary Microsoft store apps.

changedns.bat

Configures network adapters to use high-speed, privacy-focused DNS.

disable_edge_backround.bat

Prevents Edge processes from lingering in the background.

disable_suggestions.bat

Disables "Tips and Suggestions" and ads in the UI.

disable_wencryption.bat

Disables NTFS Write Encryption for better I/O performance.

edge_vanisher.ps1

Strips deep integrations of Microsoft Edge from the OS.

enablelongspaths.bat

Removes the legacy 260-character path limit.

execution policy.bat

Sets PowerShell Execution Policy to Bypass for scripts.

harden ACLs.bat

Tightens Access Control Lists on critical system files.

lats.bat

Disables "Last Access Time" updating to reduce disk activity.

openasar_installer.bat

Installs OpenAsar for a faster, lighter Discord.

optimizations.bat

General system responsiveness and priority tweaks.

privacy-script.bat

Disables telemetry, data collection, and tracking services.

registry_optimizations.bat

Modifies Registry for faster menu delays and window snapping.

shutup_edge.bat

Silences Edge update tasks, shortcuts, and nags.

win11debloat.ps1

Integrates Raphire's Win11Debloat script.

winutil.ps1

Integrates Chris Titus Tech's WinUtil toolbox.
