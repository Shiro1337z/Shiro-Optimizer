Shiro Optimizer Documentation

Shiro Optimizer is a powerful, open-source utility designed to optimize, debloat, and secure Windows 10 and 11. It combines batch scripts, PowerShell automation, and trusted third-party tools into a unified interface.

Recommended Usage

For the best experience, run the script immediately after a clean installation of Windows. Always create a System Restore Point before applying deep optimizations.

Core Tweaks

The tweaks/ directory contains granular scripts. Below is a breakdown of what each module does.

basicstuff.bat
Applies fundamental Explorer preferences, visual effects, and disables sticky keys.
privacy-script.bat
Disables telemetry, data collection services, and advertising ID tracking.
optimizations.bat
General system responsiveness tweaks, priority adjustments, and power plan settings.
registry_optimizations.bat
Modifies registry for faster menu delays, improved window snapping, and UI responsiveness.

Debloat & Remove

Shiro aggressively removes pre-installed OEM bloatware.

  • Edge Vanisher: Completely strips Microsoft Edge integrations.
  • OneDrive Removal: Uninstalls OneDrive and removes file explorer integration.
  • Cortana: Disables Cortana and removes it from the taskbar.
powershell -Command "& {Get-AppxPackage *Microsoft.YourPhone* | Remove-AppxPackage}"

Security Tools

Shiro includes a "Second Opinion" scanner downloader. These tools do not run in the background but are excellent for on-demand scanning.

Malwarebytes
Industry standard for removing malware, ransomware, and other advanced threats.
HitmanPro
Cloud-assisted scanner that acts as a second opinion if your main antivirus fails.
AdwCleaner
Specialized in removing adware, potentially unwanted programs (PUPs), and browser hijackers.

Troubleshooting

If you encounter issues after optimization, Shiro provides revert scripts in the addons/revert folder.

  • Network Issues: Run reset_dns.bat to restore default DNS settings.
  • Store Apps: Use wsreset.exe if the Microsoft Store becomes unresponsive.
