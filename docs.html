Documentation

Welcome to the comprehensive documentation for Shiro Optimizer. This toolkit provides a modular, transparent, and powerful way to optimize Windows 10 and 11.

⚠️ Critical Warning

The scripts in this toolkit modify core Windows system files, Registry keys, and services. Always create a System Restore Point before running any tweak. We are not responsible for any system instability.

Quick Start

Follow these steps to safely apply optimizations to your system.

1. Prerequisites

  • Windows 10 (20H2 or later) or Windows 11
  • Administrator privileges
  • Internet connection (for downloading dependencies)

2. Installation

PowerShell 
# Clone the repository
git clone https://github.com/Shiro1337z/Shiro-Optimizer.git
cd Shiro-Optimizer

# Install dependencies
pip install -r requirements.txt

# Run the optimizer
python main.py

Project Structure

Shiro is built with modularity in mind. Each tweak is a standalone script located in the tweaks/ directory.

Shiro-Optimizer/ ├── addons/ # Python helper modules (UI, Logic) ├── configs/ # Preset JSON configurations ├── docs/ # This documentation ├── tweaks/ # THE CORE SCRIPTS │ ├── basicstuff.bat # UAC & basic settings │ ├── Bloatware.bat # Massive app removal script │ ├── privacy-script.bat # privacy.sexy integration │ ├── registry_optimizations.bat # UI & Context Menu tweaks │ └── win11debloat.ps1 # External tool wrapper └── main.py # Main GUI entry point

Basic System Tweaks

The tweaks/apply_basic_settings.bat script handles foundational Windows settings that improve usability and responsiveness.

Feature Action Taken
User Account Control (UAC) Disables UAC completely (EnableLUA = 0) and suppresses admin consent prompts.
SmartScreen Disables SmartScreen filter for Windows Explorer and Microsoft Edge to prevent "Unrecognized App" warnings.
Fast Startup Disables Hiberboot (Fast Startup) to ensure clean shutdowns and prevent hibernation file bloat.
Mouse Acceleration Disables "Enhance Pointer Precision" by zeroing out mouse curves for 1:1 raw input (better for gaming).
Windows.old Attempts to remove the backup folder from previous Windows installations to free up space.

Bloatware Removal

The tweaks/remove_bloatware.bat script is an aggressive cleaner. It targets pre-installed OEM software, sponsored apps, and unwanted Microsoft integrations.

Apps Targeted

The script iterates through a hardcoded list of Appx packages to remove:

  • Sponsored: Candy Crush, Netflix, Disney Magic Kingdoms, Spotify, Facebook, Twitter, LinkedIn.
  • Microsoft Junk: 3D Viewer, Mixed Reality Portal, Solitaire Collection, Bing News/Weather/Sports/Finance, Get Help, Feedback Hub.
  • System Integrations: OneDrive (Full uninstall + folder cleanup), New Outlook (pre-installed web wrapper), Skype.

Prevention Mechanisms

It doesn't just remove apps; it ensures they stay gone:

Registry Keys Modified
# Prevent silent background installation of apps
HKCU\...\ContentDeliveryManager\SilentInstalledAppsEnabled = 0

# Disable "Suggested Apps" in Start Menu
HKLM\...\CloudContent\DisableWindowsConsumerFeatures = 1

# Disable Auto-Download in Windows Store
HKLM\...\WindowsStore\AutoDownload = 2

Privacy & Telemetry

We use a combination of custom scripts and the renowned privacy.sexy framework (via tweaks/apply_privacy_settings.bat) to lock down data collection.

Network Blocklist

The script appends over 100 domains to your hosts file, routing them to 0.0.0.0. This effectively creates a system-wide blackhole for:

  • watson.telemetry.microsoft.com (Error Reporting)
  • vortex.data.microsoft.com (Diagnostics)
  • ads1.msn.com (Advertising)
  • settings-ssl.xboxlive.com (Xbox Analytics)

Service Hardening

The following services are stopped and disabled:

  • DiagTrack: Connected User Experiences and Telemetry.
  • dmwappushservice: WAP Push Message Routing Service (used for telemetry).
  • WMPNetworkSvc: Windows Media Player Network Sharing Service.

Data Clearing

The script actively purges logs and history caches:

  • Clears Event Logs related to diagnostics.
  • Deletes Prefetch files to reset application launch tracking.
  • Wipes Recent Files lists for Explorer, Office, Paint, and Media Player.
  • Removes Visual Studio telemetry dumps.

Registry & UI Optimizations

Visual and functional tweaks are handled by tweaks/apply_registry_optimizations.bat.

Taskbar & Start Menu

  • Windows 11: Restores the "Classic" context menu (removing the "Show more options" click).
  • Taskbar: Aligns icons, forces "End Task" availability in context menu, and shows seconds in the system clock.
  • Clean Layout: Removes default pinned items (Edge, Store, Mail) from the taskbar for new users.

Visuals

  • Sets a unified Accent Color (Purple/Blue: #6B69D6).
  • Disables Lock Screen Spotlight (random wallpapers and quizzes).
  • Disables Transparency Effects for a slight performance gain.

Network & DNS

The tweaks/configure_dns.bat script changes your network adapter's DNS settings.

By default, it applies the following high-privacy DNS resolvers:

  • Primary: 174.138.21.128
  • Secondary: 188.166.206.224

Note: You can edit the script to use Cloudflare (1.1.1.1) or Google (8.8.8.8) if preferred.

Context Menu Tools

Shiro injects powerful tools directly into your Desktop right-click menu via Registry keys.

1. Safe Mode Switcher

Right-click your desktop to instantly reboot into:

  • Safe Mode
  • Safe Mode with Networking
  • Safe Mode with Command Prompt
  • Normal Mode (to exit Safe Mode)

2. Power Plan Switcher

Switch power profiles without digging through Control Panel:

  • Power Saver: Max battery life.
  • Balanced: Default Windows behavior.
  • High Performance: Keeps CPU clock high.
  • Ultimate Performance: Reduces micro-latencies (enabled by script).

Troubleshooting

"Access Denied" Errors

Ensure you are running main.py or the batch scripts as Administrator. Many Registry keys are protected.

Internet Not Working

If the DNS change fails or blocks too much, run:

netsh int ip reset
ipconfig /flushdns

Reverting Changes

If you created a System Restore Point as advised, simply restore to that point. Shiro currently does not have a universal "Undo" button due to the destructive nature of debloating (deleted apps cannot be undeleted easily).
